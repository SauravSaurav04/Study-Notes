# Study-Notes

A personal repository for organizing and maintaining study notes across multiple subjects and learning areas. Each note is stored as a separate markdown file for easy review, editing, and future expansion.

## Organization

- Use subject or course folders for different topics
- Store class notes, summaries, practice prompts, and roadmaps as markdown files
- Keep each note focused and easy to update

> Note: This repository is meant for general study notes and can grow beyond any single subject.
